This is a Excel VBA project for Log Select and Analysis
In this case, I get a log file with Hexadecimal format, need to know each line with key word.

Files:
    1. Excel document: xxx.xlsm
    2. Excel VBA code file

How to use:
    1.Open xxx.xlsm, press key alt+F8 to run VBA macro script in office excel with VBA support,
      to view how does it run.
    2.Modify the keywords as your need,and input your own log.
